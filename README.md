🛒 DevStore – Streamlit E-Commerce & Analytics Dashboard
🚀 Overview

DevStore is a modern, interactive E-Commerce Web Application built using Streamlit, combining:

🛍️ Shopping System
🧾 Cart Management
📊 Analytics Dashboard

This project demonstrates how Python can be used to build full-stack-like applications with clean UI and real-time interactivity.

✨ Features
🛍️ E-Commerce
Product selection with pricing
Add to cart functionality
Quantity management
Checkout system
📊 Dashboard
KPI Cards (Sales, Orders, Customers)
Sales graph visualization
Weekly analytics
🎨 UI/UX
Dark theme modern UI
Custom CSS styling
Card-based layout
Responsive design
🧠 Tech Stack
Frontend + Backend: Streamlit
Data Handling: Pandas, NumPy
Visualization: Plotly / Streamlit Charts
State Management: Streamlit Session State
Styling: HTML + CSS

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/your-username/devstore.git
cd devstore
2️⃣ Install Dependencies
pip install streamlit pandas numpy plotly
3️⃣ Run Application
streamlit run devstore.py
